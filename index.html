Portfolio

  • Ryunco Ecommerce

    Ryunco Ecommerce adalah aplikasi platform e-commerce yang dibangun menggunakan CodeIgniter 3....... Aplikasi ini dirancang untuk menyediakan pengalaman belanja online yang mudah dan efisien. Dengan tampilan yang responsif dan fitur lengkap seperti katalog produk, keranjang belanja, sistem pembayaran, dan manajemen pesanan, Ryunco Ecommerce bertujuan untuk memudahkan pengguna dalam melakukan transaksi online. Aplikasi ini juga digunakan sebagai bagian dari ujian mata kuliah Web Development, yang menguji kemampuan dalam pengembangan aplikasi berbasis web dengan framework CodeIgniter 3.

  • E-cuti

    Website E-Cuti merupakan aplikasi berbasis web yang dikembangkan untuk mempermudah proses pengajuan ...... dan pengelolaan cuti bagi pegawai di lingkungan Suku Dinas Pendidikan Wilayah 2 Jakarta Pusat. Sistem ini dibangun menggunakan framework CodeIgniter 3 dan bahasa pemrograman PHP, serta menggunakan database MySQL untuk menyimpan data. Aplikasi ini dirancang untuk menggantikan proses pengajuan cuti secara manual, sehingga menjadi lebih cepat, efisien, dan terorganisir. Fitur utama meliputi pengajuan cuti oleh pegawai, verifikasi dan persetujuan cuti oleh atasan, serta rekapitulasi data cuti. Sistem juga dilengkapi dengan hak akses berbeda sesuai peran pengguna, seperti pegawai, atasan, dan admin. Pengembangan website E-Cuti ini merupakan bagian dari penyusunan skripsi sebagai syarat untuk menyelesaikan studi pada jenjang Strata 1 (S1), dengan fokus pada pemanfaatan teknologi web untuk mendukung sistem administrasi kepegawaian yang modern dan efektif.

  • Architect

    Architect Connect adalah sebuah platform digital yang menghubungkan langsung antara arsitek profesional dan ......calon pelanggan. Melalui website ini, pengguna dapat mencari, melihat portofolio, dan berinteraksi dengan arsitek sesuai kebutuhan proyek mereka—baik untuk rumah tinggal, kantor, maupun bangunan komersial lainnya. Website ini dibangun menggunakan ReactJS untuk tampilan antarmuka yang interaktif dan responsif, serta NodeJS sebagai backend untuk menangani proses autentikasi, manajemen data, dan komunikasi antara pengguna dan arsitek secara real-time. Dengan fitur pencarian, chat langsung, ulasan proyek, dan sistem pemesanan, Architect Connect mempermudah proses kolaborasi dari awal hingga proyek selesai.
